\# Speech-to-Text Converter



\## Overview



Speech-to-Text Converter is a web-based application that converts spoken audio into text using speech recognition technology. The application provides a simple and user-friendly interface for capturing voice input and displaying the corresponding text output.



\## Features



\* Convert speech into text in real time

\* Easy-to-use web interface

\* Fast and accurate speech recognition

\* Supports voice input through a microphone

\* Responsive design



\## Technologies Used



\* Python

\* Flask

\* HTML

\* CSS

\* JavaScript

\* Web Speech API

\## Installation



1\. Clone the repository:



```bash

git clone https://github.com/vennelakongara2307/Speech-to-text-Converter.git

```



2\. Navigate to the project directory:



```bash

cd Speech-to-text-Converter

```



3\. Install dependencies:



```bash

pip install flask

```



4\. Run the application:



```bash

python app.py

```



5\. Open your browser and visit:



```text

http://127.0.0.1:5000

```
